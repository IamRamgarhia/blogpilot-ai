# Featured Snippet Targeting

Featured snippets come in three formats: paragraph, list, and table. Choose the right format:

| Snippet format | When to use | First-paragraph rule |
|---|---|---|
| Paragraph | "What is X", "Why does X" | 40-55 words, direct answer in first sentence |
| Numbered list | "How to X", "Steps to X" | Intro 1-2 sentences then 4-8 numbered steps |
| Table | "X vs Y", "best X by Y" | Title sentence then a 3-5 column table |

## Rules

1. **First sentence directly answers the question.** No preamble. No "Well, ..."
2. **40-55 words** is the empirical sweet spot Google extracts most often.
3. Use the **exact question phrasing** in either the H2 or first sentence.
4. **Define the key term** in the snippet itself. Google does not read 3 paragraphs down for context.
5. For lists: start each item with the action verb in present tense.
